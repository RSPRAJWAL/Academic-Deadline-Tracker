# Roadmap

## Version 1.0 (Current) - Desktop Application

### Core Features
- [x] Add/edit/delete tasks
- [x] Task attributes: title, description, due date, course, priority, reminder
- [x] List view with countdowns
- [x] Search and filter by course and priority
- [x] Mark tasks as completed
- [x] Export/import tasks as JSON
- [x] SQLite database storage
- [x] Local desktop notifications
- [x] Firebase Cloud Messaging integration (basic)
- [x] Simple Tkinter GUI
- [x] Unit tests for core logic

### UI Features
- [x] Left sidebar for filters
- [x] Main list view
- [x] Right panel for task details
- [x] Responsive layout

### Technical Features
- [x] Clean, modular code with docstrings
- [x] Type hints
- [x] Requirements.txt and README.md
